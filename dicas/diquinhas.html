O atributo pode ter os seguidores valores: - true ou uma string vazia, que indica que o elemento é editável. - false, o que indica que o elemento não é editável. Embora seus valores permitidos incluam true ou false, esse atributo é enumerado e não booleano. Dessa forma, somente o atributo 'contenteditable' é valido como true.
