​A tag datalist especifica uma lista de opções predefinidas para um elemento input. O atributo id do elemento datalist deve ser igual ao atributo list do elemento input (isso os vincula).
O atributo 'contenteditable' indica se o elemento pode ser editável pelo usuário.

O atributo pode ter os seguidores valores: - true ou uma string vazia, que indica que o elemento é editável. - false, o que indica que o elemento não é editável. Embora seus valores permitidos incluam true ou false, esse atributo é enumerado e não booleano. Dessa forma, somente o atributo 'contenteditable' é valido como true.
O atributo 'draggable' indica se o elemento pode ser arrastado ou não.
